## Controller

- **Movimiento:**
  - <kbd>←</kbd>: Moverse hacia la izquierda.
  - <kbd>→</kbd>: Moverse hacia la derecha.
  - <kbd>Tecla "TAB"</kbd>: Saltar.
  - <kbd>Tecla "CTRL"</kbd>: Agacharse.
